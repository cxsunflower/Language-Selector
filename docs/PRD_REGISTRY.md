# PRD 总集

| 版本 | 标题 | 需求内容（详细摘要） | PRD 链接 |
|---|---|---|---|
| PRD-001 | Language Selector 单应用语言切换 | 目标是为 Android 13 及以上、缺少官方 App Language 入口的设备提供一个可操作的单应用语言管理前端。范围覆盖能力准备、应用浏览与搜索、单应用语言设置、固定常用语言以及快捷设置磁贴快速切换。关键规则包括：仅负责写入目标应用 Locale，不负责翻译；默认隐藏未修改的系统应用；支持恢复系统默认；固定语言会同步进入磁贴轮换集合。异常边界包括：Shizuku 或 Root 不可用时必须阻塞进入可管理态；系统应用与不支持目标语言的应用不作为稳定承诺对象；磁贴在无固定语言、前台系统应用或服务不可用时必须显示不可用。非目标是不处理低于 Android 13 的兼容方案，也不扩展为应用翻译器。 | docs/prd/PRD-001.md |
